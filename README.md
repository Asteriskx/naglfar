# Run

## Requisites
- Latest Rust (recommend [rustup](https://www.rustup.rs/))
- GTK (for gtk-rs)

## Build & Run

By default, Naglfar shows `example/test.html` applied `example/test.css`.

```sh
$ cargo run
$ # or...
$ cargo run --release
```
